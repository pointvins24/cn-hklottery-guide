# cn-hklottery-guide

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态 HTML 文件形式存在，不针对任何特定域名或网站，仅供浏览和参考。

## 项目简介

cn-hklottery-guide 是一个静态页面集合仓库，主要存放与活动信息、流程说明、指引等相关的 HTML 文件。所有页面均为独立文件，不依赖后端服务，可直接在浏览器中打开查看。

## 目录说明

```
cn-hklottery-guide/
├── index.html          # 主入口页面（如有）
├── pages/              # 存放主要 HTML 页面
├── assets/             # 静态资源（样式、图片、脚本等）
└── README.md           # 本文件
```

- `pages/`：包含归档的 HTML 页面，每个文件为一个独立页面。
- `assets/`：存放页面引用的 CSS、JavaScript、图片等资源文件。

## 页面归档说明

仓库中的 HTML 页面均为历史或当前使用的静态页面归档。每个页面均以独立文件形式保存，便于本地查看、分享或二次修改。

- 页面内容可能包含通用指引、说明或流程展示。
- 不包含任何动态数据或外部接口调用。
- 所有页面均为纯前端实现，无后端依赖。

## 维护说明

- 本仓库为归档性质，不提供实时更新或维护承诺。
- 如有需要，可自行 Fork 或下载使用。
- 页面内容如涉及外部引用，请自行核实其可用性与安全性。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 直接在浏览器中打开 `index.html` 或 `pages/` 目录下的任意 HTML 文件。
3. 所有页面均为自包含，无需额外配置。
